My Projects

Through side projects, university coursework, and internships, I have completed many projects.
Here are some of my favorites.

Pantry Planner

Created a lightweight full stack web application that matches an inventory of ingredients with possible recipes. The back end was built using Express.js on top of Node.js and connected to a MongoDB database. The front end was developed using React.js.
QA Toolkit

Programmed an interactive GUI using the tkinter Python library. The GUI audits and compiles disk usage statistics on a distributed file system. The program reports disk space metadata on a per-user basis including total used disk space and stale data.

Reading Log

Developed a reading log application for Android devices. The app (developed mainly in Java) stored user reading info in a database accessed using SQLite and allowed users log individual reading sessions unique to each book.

Player Predictions

Developed a supervised machine learning model that predicts the position a NBA player is listed at based on their box-score statistics. Using sklearn's dynamic classification library, the final model implemented a voting classifier trained on over 24,000 records using a variety of finely-tuned base classifieres.

Rijndael Cipher

Implemented the Advanced Encryption Standard (or Rijndael) cipher in Java. This cipher is one of the most widespread symmetric encryption algorithms, and it is largely responsible for secure Internet communication.
